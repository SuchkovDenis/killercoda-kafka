# Kafka CLI

В этом сценарии мы освоим основные CLI-команды для работы с топиками Kafka: создание, просмотр, изменение и удаление.

## Загрузка окружения

После нажатия **Start** в терминале (справа внизу) вы увидите:

```
=============================================
  Kafka-кластер загружается...
=============================================
```

Это нормально — идёт скачивание Docker-образов и запуск 3 брокеров Kafka. **Обычно занимает 1-3 минуты.** Пока ждёте, прочитайте описание задания ниже.

Когда появится сообщение **«✅ Kafka-кластер готов!»** — можно приступать.
