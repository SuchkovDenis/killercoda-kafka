# Распределение по партициям

Обратите внимание на вывод:
- Каждый пользователь всегда в **одной** партиции (ключ = user_id)
- Разные пользователи могут быть в одной партиции (коллизия хешей)
- Распределение не обязательно равномерное

Посмотрите подробнее через Kafka UI:

[Открыть Kafka UI]({{TRAFFIC_HOST1_8080}})

> Kafka UI может загружаться 30-60 секунд — это нормально, дождитесь появления интерфейса.

Перейдите в Topics → user-events → Messages и отфильтруйте по ключу.
