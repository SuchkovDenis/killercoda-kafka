# Apache Kafka — интерактивные лабораторные работы

Интерактивные сценарии для курса Apache Kafka на [Stepik](https://stepik.org/course/217675).

Каждый сценарий — это практическая лабораторная работа, которую можно выполнить прямо в браузере через [KillerCoda](https://killercoda.com).

## Как использовать

1. Перейдите по ссылке на нужный сценарий
2. Нажмите "Start Scenario"
3. Следуйте инструкциям, выполняя команды в терминале справа
4. Каждый шаг проверяется автоматически
