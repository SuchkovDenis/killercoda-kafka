# Kafka UI

Откройте Kafka UI в браузере:

[Открыть Kafka UI]({{TRAFFIC_HOST1_8080}})

> Kafka UI может загружаться 30-60 секунд — это нормально, дождитесь появления интерфейса.

В интерфейсе вы увидите:
- **Brokers** — три брокера кластера
- **Topics** — созданный топик `test`
- Кликните на топик, чтобы увидеть партиции и их распределение

Kafka UI — удобный инструмент для мониторинга кластера, просмотра сообщений и управления топиками.
